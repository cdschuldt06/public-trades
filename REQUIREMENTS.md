# PublicTrades Requirements

## Product Goal

PublicTrades is a full-stack web app that collects publicly disclosed congressional stock trades and lets users search and analyze those disclosures.

## MVP Features

### 1. Recent Trades Page

Users can view a table of recent congressional trades.

Displayed fields:
- Politician name
- Chamber
- Party
- State
- Ticker symbol
- Asset name
- Transaction type
- Amount range
- Transaction date
- Disclosure date

### 2. Search and Filters

Users can filter trades by:
- Politician name
- Ticker symbol
- Transaction type

### 3. Politician Detail Page

Users can view a single politician’s trading history.

The page should show:
- Politician name
- Chamber
- Party
- State
- Total reported trades
- Trade history table

### 4. Manual Data Import

For the MVP, trade data can be imported manually from a local CSV or JSON file.

## Not Included in MVP

- Real-money trading
- Paper trading
- Portfolio simulator
- User accounts
- Email alerts
- Paid subscriptions
- AI predictions
