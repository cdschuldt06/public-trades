# PublicTrades Development Roadmap

## Phase 0: Planning

Status: In Progress

Goals:
- Define MVP requirements
- Define architecture
- Define database schema
- Define API contract
- Set up GitHub repository

## Phase 1: Backend Foundation

Goals:
- Initialize Node/TypeScript backend
- Add Express server
- Add `/health` endpoint
- Add local development scripts

## Phase 2: Database Foundation

Goals:
- Add PostgreSQL
- Add Prisma
- Create initial Prisma schema
- Run first migration

## Phase 3: Seed Data

Goals:
- Create sample trade data
- Import sample politicians, tickers, and trades
- Verify data can be queried from the backend

## Phase 4: Trade API

Goals:
- Add `GET /trades`
- Add filtering by politician, ticker, and transaction type

## Phase 5: Angular Frontend

Goals:
- Initialize Angular app
- Create recent trades page
- Connect frontend to backend

## Phase 6: Politician Pages

Goals:
- Add politician list page
- Add politician detail page
- Display politician trade history

## Future Phases

- Portfolio simulator
- Paper trading
- Automated disclosure ingestion
- Real-money trading research
